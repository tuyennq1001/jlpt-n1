# 📚 Kho Tài Liệu Ôn Luyện Ngữ Pháp JLPT N1

Chào mừng bạn đến với kho tài liệu ôn luyện ngữ pháp **JLPT N1**. Tại đây, các ngữ pháp nâng cao được nhóm lại theo các từ gốc hoặc nhóm ý nghĩa để giúp bạn dễ dàng so sánh, đối chiếu và tránh nhầm lẫn trong bài thi.

**🔥 Tổng số mẫu ngữ pháp: 115 mẫu** (chia thành 11 nhóm gốc cơ bản và 13 nhóm ý nghĩa thực tế).

---
